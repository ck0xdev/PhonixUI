Overview

Introduction

PhonixUI is a high-fidelity, zero-dependency CSS framework designed for speed, professional aesthetics, and modern web standards. Built completely from scratch utilizing OKLCH color science.

Framework Version V1.0

This documentation reflects the official stable release of PhonixUI. It covers everything from basic layouts to advanced GPU-accelerated glow and glass effects.

Installation

Get up and running in seconds. PhonixUI is a single CSS file with zero build tools required.

HTML 
<!-- 1. Download phonixUI.min.css and place it in your project -->
<!-- 2. Link it in the <head> of your document -->

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My App</title>
  <link rel="stylesheet" href="phonixUI.min.css">
</head>
<body class="theme-blue">
  <!-- Start building! -->
</body>
</html>

Theming (OKLCH Engine)

PhonixUI uses the modern OKLCH color space. By changing a single CSS variable (--ph-base-hue), the entire framework palette (backgrounds, surfaces, borders, primary accents, and glows) recalculates automatically.

theme-blue theme-green theme-red theme-purple theme-orange
HTML 
<!-- Apply a built-in theme class to the body -->
<body class="theme-purple">

<!-- OR use a completely custom hue (0-360) inline -->
<body style="--ph-base-hue: 300;">

Grid System

A mobile-first, 12-column CSS Grid. Classes mimic standard flex grids but use native CSS display: grid for absolute precision. Breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px).

col-span-12
md-col-span-6
md-col-span-6
col-span-4
col-span-8
HTML 
<div class="grid">
  <!-- Full width on mobile, half width on medium screens -->
  <div class="col-span-12 md-col-span-6">Left Column</div>
  <div class="col-span-12 md-col-span-6">Right Column</div>

  <!-- 1/3 and 2/3 split on all screens -->
  <div class="col-span-4">Sidebar</div>
  <div class="col-span-8">Main Content</div>
</div>

Flexbox & Spacing

PhonixUI includes a robust utility engine for alignment, layout, and spacing without writing custom CSS.

Flex Utilities

.d-flex, .flex-column, .justify-content-[start|center|between|end], .align-items-[start|center|end].

Spacing Scale

Format: {property}-{direction}-{size}.
Properties: m (margin), p (padding).
Directions: t (top), b (bottom), l (left), r (right), x (horizontal), y (vertical).
Sizes (px): 0, 2, 4, 8, 10, 12, 16, 20, 24, 32, 40, 48, 64, 80, 100, 120.

HTML 
<!-- Flex container, centered vertically, space between horizontally, gap 16px -->
<div class="d-flex align-items-center justify-content-between gap-16">
  
  <!-- Margin-bottom 24px, Padding-horizontal 32px -->
  <div class="m-b-24 p-x-32">Item</div>

</div>

Buttons

Highly polished interactive elements. Buttons include transition states, focus-visible rings for accessibility, and active press states.

HTML 
<!-- Standard Primary -->
<button class="btn btn-primary">Primary</button>

<!-- Primary with GPU Glow Effect -->
<button class="btn btn-primary ph-btn-glow">Glow Active</button>

<!-- Glassmorphic Button -->
<button class="btn btn-glass">Glass</button>

<!-- Sizes -->
<button class="btn btn-primary btn-sm">Small</button>
<button class="btn btn-primary btn-lg">Large</button>

Cards

Base surface containers for grouping related content. Combine with interaction utilities for dynamic 3D lift effects.

Standard Card

A solid surface container with predefined borders, radii, and structural padding (header, body, footer).

✨
Interactive Card

Hover over this card. It utilizes .ph-card-interactive to trigger a 3D translateY lift and inject a primary-colored glow shadow.

HTML 
<!-- Interactive Card Structure -->
<div class="card ph-card-interactive">
  <div class="card-header font-bold">Title</div>
  
  <p class="text-muted text-sm">Card body content goes here.</p>
  
  <div class="card-footer">
    <button class="btn btn-primary w-100">Submit</button>
  </div>
</div>

Forms & Inputs

Fully styled, accessible form controls. Inputs seamlessly adapt to the OKLCH theme engine for focus rings.

Username is already taken.
HTML 
<!-- Text Input -->
<div class="form-group">
  <label class="label">Email Address</label>
  <input type="email" class="input" placeholder="user@domain.com">
</div>

<!-- Select Dropdown -->
<select class="select">
  <option>Option 1</option>
</select>

<!-- Error State (triggered by aria-invalid) -->
<input type="text" class="input" aria-invalid="true">
<div class="form-error">Error message.</div>

Glass & Glow FX

PhonixUI's signature aesthetic relies on CSS variables combined with backdrop-filter and complex layered box-shadows. These are GPU-accelerated.

Glass Component

Notice the blur effect over the ambient glow background.

HTML 
<!-- Ambient Background Glow -->
<div class="ph-glow-primary absolute top-0 left-0"></div>

<!-- Glass Surface -->
<div class="ph-glass p-32 rounded-md">
  Frosted content over background elements.
</div>

Tooltips

Pure CSS tooltips using the data-tooltip attribute. No Javascript initialization required.

U
HTML 
<button class="btn" data-tooltip="Tooltip text here">Hover me</button>

Skeleton Loaders

Animated shimmer placeholders for asynchronous content loading states.

HTML 
<!-- Apply .skeleton to empty divs and set dimensions -->
<div class="skeleton rounded-full" style="width: 48px; height: 48px;"></div>
<div class="skeleton m-t-16" style="height: 16px; width: 100%;"></div>
